
# S3 Bucket Versioning Setup

## Objective
Enable versioning on an existing S3 bucket to track and manage multiple versions of objects.

## Tasks Completed
1. Selected the S3 bucket to configure versioning.
2. Enabled versioning and confirmed it was successfully activated.
