# AWS S3 Bucket Setup

## 📌 Project Overview

This project demonstrates how to create an Amazon S3 bucket and upload an object to it.

Amazon S3 (Simple Storage Service) is an object storage service that allows users to store and retrieve any amount of data from anywhere on the web.

This lab covers the basic operations of creating a bucket and uploading objects, which are fundamental tasks when working with cloud storage.
